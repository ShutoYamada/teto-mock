# ゲーム性仕様

テトリス+ローグライクゲームのゲームシステムに関する仕様です。

## 全体構成

- ダンジョン画面と戦闘画面から成る
- ダンジョンは3ステージ構成
- 1ステージごとにスタート地点からツリー状に分岐する。どの分岐ルートを選んでも最後はステージのボスにたどり着く
- ボスを倒すと次のステージに進む
- 3ステージ目のボスを倒すとゲームクリア
- ボスに負けるとゲームオーバー

---

## ダンジョン

- 各ステージごとに1〜3のスタート地点があり、プレイヤーは任意のスタート地点を選ぶ
- スタート地点からツリー状に分岐する
- 各分岐ごとに「戦闘イベント」や「ランダムイベント」が発生する
  - 「戦闘イベント」: 敵と戦闘する
  - 「ランダムイベント」: アーティファクトを獲得する

---

## 戦闘システム

### ターン進行

- プレイヤーターンと敵ターンが交互に進む
- 戦闘は必ずプレイヤーターンから始まる
- プレイヤーが手動でターン終了を宣言すると敵ターンになり、敵の攻撃を受ける

### プレイヤーステータス

- ゲーム開始時のHP: 50、MP: 5
- 敵の攻撃を受けるとHPが減る。HPが0になるとゲームオーバー

### カードとMP

- プレイヤーはデッキからランダムに配られた手札(初期7枚)から1枚を選び、テトリスフィールド(デフォルト7×7)に配置する
- カードにはコストがあり、プレイヤーはコスト分のMPを消費してカードを配置する
- MPは毎ターン開始時に全回復する

### ダメージ計算

- カードには基礎攻撃力があり、配置時に敵にダメージを与える
- テトリスフィールドの各行・列には行列攻撃力があり、対象の行・列を消すと敵にダメージを与える
- 連続してラインを消すとコンボとなり、コンボ数に応じて追加ダメージを与える

### 敵の行動

- 敵の攻撃はあらかじめ数値として表示される
- 敵ターンにそのダメージを受ける

---

## デッキ

- ゲーム開始時のデッキはI, O, T, S, Z, J, Lの基本カードを各2枚(計14枚)
- 戦闘勝利後にリザルト画面でカードが3枚ランダムに提示され、1枚をデッキに加えられる
- 戦闘勝利後に提示されるカードは基本カードと特殊ミノカードの混合

---

## 特殊ミノカード

- 十字・H字などの特殊な形状のミノ
- ブロック中に特殊ブロックを含む場合がある
- 攻撃力やコストが通常カードと異なる

### 特殊ブロック

| 種別         | 効果                                            |
| ------------ | ----------------------------------------------- |
| 爆弾ブロック | 消滅時に周囲8マスを消しつつ追加ダメージを与える |
| 剣ブロック   | 配置時の基礎ダメージが上昇する                  |
| 盾ブロック   | 次の敵ターンに受けるダメージを軽減する          |

---

## アーティファクト

- 入手することで永続的な効果や、任意のタイミングで使える特殊スキルを得る

### アーティファクトの例

| 名称     | 効果                                    |
| -------- | --------------------------------------- |
| 勇者の剣 | すべてのカードの基礎攻撃力上昇          |
| 鉄壁の盾 | 受けるダメージを軽減する                |
| 風の剣   | ターン開始時に敵全体に3ダメージを与える |

---

## 貨幣 (Gold)

- 戦闘勝利時に一定量のGoldを獲得できる
