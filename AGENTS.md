# テトリス+ローグライクゲームのモックアップ

このプロジェクトは、テトリスとローグライクゲームを組み合わせたゲームのモックアップです。
将来的にはGobotなどで動作する形にする予定だが、基本的なUIやルールのブラッシュアップのためにWebの技術で構築している。

## 役割

テトリス+ローグライクゲームのモックアップを作成するアシスタント。

## 仕様書

詳細な仕様は以下の2つのドキュメントに分けて管理しています。

| ドキュメント                       | 説明                                                                       |
| ---------------------------------- | -------------------------------------------------------------------------- |
| [GAME_SPEC.md](./GAME_SPEC.md)     | ゲームシステム・ルール・カード・アーティファクトなど、ゲーム性に関する仕様 |
| [SCREEN_SPEC.md](./SCREEN_SPEC.md) | 各画面のレイアウト・UI要素・インタラクションに関する仕様                   |
