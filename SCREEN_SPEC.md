# 画面仕様

テトリス+ローグライクゲームの各画面に関するUI/UX仕様です。

## 画面一覧

| 画面名             | 説明                                                 |
| ------------------ | ---------------------------------------------------- |
| ダンジョン画面     | ステージのルートマップを表示し、次の進行先を選択する |
| 戦闘画面           | テトリスフィールドでカードを配置して敵と戦う         |
| リザルト画面       | 戦闘勝利後の報酬選択画面                             |
| ゲームオーバー画面 | 敗北時に表示される画面                               |

---

## ダンジョン画面

- ツリー状のルートマップを表示する
- 各ノードは「戦闘イベント」または「ランダムイベント」のいずれかを示す
- プレイヤーが進行先ノードをタップ/クリックすることで次のイベントへ移行する
- ステージ開始時、スタート地点が1〜3つ表示され、プレイヤーが選択する

---

## 戦闘画面

### レイアウト構成要素

| 要素                     | 説明                                                                                    |
| ------------------------ | --------------------------------------------------------------------------------------- |
| テトリスフィールド       | 7×7（デフォルト）の盤面。カードを配置するエリア                                         |
| デッキ情報表示           | 現在の「山札(Deck)」と「捨て札(Discard)」の枚数を表示                                   |
| 手札エリア               | 現在の手札を表示する                                                                    |
| プレイヤーステータス表示 | HP・MPを表示する                                                                        |
| 敵情報表示               | 敵の数だけHP・次ターンの予告ダメージを表示する。<br/>現在のターゲットを視覚的に強調する |
| ターン終了ボタン         | クリックで敵ターンに移行する                                                            |

### インタラクション

- 手札からカードを選択してテトリスフィールド上に配置する
- カード選択中、特定の操作（右クリック、または指定のキー等）でカードの形状を回転できる
- 複数敵がいる場合、敵のステータス領域をクリックして攻撃対象(ターゲット)を切り替える
- 配置時にカード基礎攻撃力ダメージが即時発生する
- ラインが揃った場合、行・列の消去とダメージが発生する
- 連続消去時はコンボ表示と追加ダメージが発生する
- ターン終了ボタンを押すと敵の攻撃フェーズへ移行する

## リザルト画面

- 戦闘勝利後に表示される
- 獲得Gold量を表示する
- ランダムに選ばれたカード3枚を提示し、プレイヤーは1枚を選んでデッキに加える

---

## ゲームオーバー画面

- 戦闘敗北後に表示される
- 「最初からやり直す」などのアクションを提示する
