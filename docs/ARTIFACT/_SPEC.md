# 敵ユニット

## 基本

- すべてのユニットはレアリティ、イベント抽選獲得フラグを持つ
- レアリティはコモン、アンコモン、レア、ボスの4種類がある
- ランダムイベントの結果によってランダムでアーティファクトを獲得する場合、そのイベントごとに定められた割合でレアリティごとに異なるアーティファクトを獲得する。また、イベント抽選獲得フラグを持つアーティファクトのみが抽選対象となる
- アーティファクトはプレイヤーが任意で破棄することはできず、ランダムイベントの内容によってのみ破棄、変化する
- 既にプレイヤーが所持しているアーティファクトと同じアーティファクトは抽選対象に含まれず、同一のアーティファクトをプレイヤーが所持することはない
- アーティファクトは永続的に効果を発揮し、ランダムイベントによって破棄されない限り効果は失われない
- アーティファクトの効果はゲームの基本ルールの改変を伴うものがある(例：基本盤面のマス数増加、ダメージ計算式の変更)
