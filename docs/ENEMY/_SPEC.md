# 敵ユニット：基本仕様

## 基本

- すべてのユニットはHPを持つ。HPが固定でない場合は出現時に下限上限内でランダムに決定する
- すべてのユニットは自身が行う行動の中から1つ選択する。行動確率は行動名の後ろに(P%)で示される
- すべてのユニットは出現ステージ数を持つ
- すべてのユニットは「通常」「エリート」「ボス」のいずれかの種類を持つ
  - 「通常」は通常の敵ユニット。自身の「出現ステージ数」と同じステージ中の通常マスに出現する
  - 「エリート」は通常の敵ユニットより強い敵ユニット。自身の「出現ステージ数」と同じステージ中のエリートマスにのみ出現する
  - 「ボス」は自身の「出現ステージ数」と同じステージ中の最終ノードのみ出現する
  - いずれのユニットもイベントマスで発生する特殊イベントによって出現することがある
- ユニットはユニット自身の行動、もしくはプレイヤーの行動によって特殊状態を持つ
- 異なる種類の特殊状態は重複する。同じ種類の特殊状態は重複せず、N値は合算される
- すべての敵ユニットは「撃破時のゴールド」を持つ。プレイヤーは戦闘勝利時に、撃破したすべての敵のゴールドの合計を獲得する

### 特殊状態

- 防御(N)：Nダメージを軽減する。一度軽減が発動したら解除される
- 憤怒(N)：自身がプレイヤーに与えるダメージにNを加算する
- ダメージ反射(N)：ダメージを受けた場合、Nダメージをプレイヤーに与える
- 転倒(N)：Nターンの間、受けるダメージが1.5倍される
- 挑発(N)：Nターンの間、プレイヤーは他の敵ユニットを攻撃できず、このユニットのみを攻撃対象にできる
