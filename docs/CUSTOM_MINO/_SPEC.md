# 特殊ブロック仕様

| 種別           | レアリティ | 効果                                                                |
| -------------- | ---------- | ------------------------------------------------------------------- |
| 剣             | コモン     | 配置している間中、全てのミノカードの配置時の基礎ダメージが1上昇する |
| クロス         | コモン     | 消滅時に周囲8マスを消しつつ敵全体に10追加ダメージを与える           |
| 盾             | コモン     | 配置時に次の敵ターンに受けるダメージを1軽減する                     |
| ドロー         | コモン     | 配置時に手札を1枚引く                                               |
| マナ           | コモン     | 消滅時にMPが1回復する                                               |
| ゴールドの脈動 | コモン     | 消滅時にゴールドを5加算する                                         |
| 痛みの増資     | コモン     | ターン終了時に盤面に存在する場合、自分に1ダメージ与える             |
| スクエアボム   | レア       | 全マスが爆弾ブロック。消滅時に周囲8マス破壊&10ダメ                  |
| ワンツー       | コモン     | 消滅時にコンボ数を+1するブロックを両端に持つ                        |
| ジャブ         | コモン     | 縦2マス。コスト0。全マスがコンボ加算ブロック                        |
| 弓             | コモン     | 縦3マス。このブロックを含む行列消滅時のダメージが敵全体に及ぶ       |

## 特殊ブロック一覧

| ブロック名         | 効果                                                                     |
| ------------------ | ------------------------------------------------------------------------ |
| 爆弾ブロック       | 消滅時に周囲8マスを消しつつ敵全体に10追加ダメージを与える                |
| 剣ブロック         | 配置している間中、全てのミノカードの配置時の基礎ダメージが1上昇する      |
| 盾ブロック         | 配置時に次の敵ターンに受けるダメージを1軽減する                          |
| マナブロック       | 消滅時にMPが1回復する                                                    |
| ドローブロック     | 配置時に手札を1枚引く                                                    |
| トゲブロック       | ターン終了時に盤面に存在する場合、自分に1ダメージ与える                  |
| ボーダーブロック   | 消滅時に行消滅だった場合はダメージを5加算する                            |
| ストライプブロック | 消滅時に列消滅だった場合はダメージを5加算する                            |
| ゴールドブロック   | 消滅時にゴールドを5加算する                                              |
| コンボ加算ブロック | 消滅時にコンボ数を+1する                                                 |
| ゴミブロック       | 隣接するマスにミノが配置された時の基礎ダメージを1マイナスする            |
| 弓ブロック         | このブロックを含む行列消滅時のダメージが敵全体に及ぶ                     |
| ライフブロック     | このブロックを含む行列消滅時にHPを3回復する                              |
| ハードブロック     | 消滅時に消滅せず、通常ブロックに置き換わる                               |
| 共鳴ブロック       | 配置時に既に配置されている共鳴ブロックの数だけ、配置時ダメージを加算する |
