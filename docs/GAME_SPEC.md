# ゲーム性仕様

テトリス+ローグライクゲームのゲームシステムに関する仕様です。

## 全体構成

- ダンジョン画面と戦闘画面から成る
- ダンジョンは3ステージ構成
- 1ステージごとにスタート地点からツリー状に分岐する。どの分岐ルートを選んでも最後はステージのボスにたどり着く
- ボスを倒すと次のステージに進む
- 3ステージ目のボスを倒すとゲームクリア
- ボスに負けるとゲームオーバー

---

## ダンジョン

- 各ステージごとに1〜3のスタート地点があり、プレイヤーは任意のスタート地点を選ぶ
- スタート地点からツリー状に分岐する
- 各分岐ごとに「戦闘イベント」や「ランダムイベント」が発生する
  - 「戦闘イベント」: 敵と戦闘する
  - 「ランダムイベント」: アーティファクトを獲得する
