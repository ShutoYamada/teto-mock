# ショップ：基本仕様

## 基本

- ショップではカード、アーティファクトの購入ができる
- カードはレアリティに応じて価格が変動する
  - コモン: 50G
  - アンコモン: 125G
  - レア: 250G
- カードは5つまでショップのラインナップとして登場し、購入すると補充されない
- アーティファクトはレアリティに応じて価格が変動する
  - コモン: 100G
  - アンコモン: 200G
  - レア: 300G
- アーティファクトは3つまでショップのラインナップとして登場し、購入すると補充されない
- アーティファクトは店売りフラグが立っているもののみがラインナップとして登場する
