# 特殊ミノカード

## 特殊ミノカード一覧

### 攻撃系

#### 十字カード

- 十字に配置
- コスト1, 攻撃力4
- 中心1マスに爆弾ブロックを持つ

#### スクエアボムカード

- 縦横3マスの正方形に配置
- コスト4, 攻撃力3
- 全マスに爆弾ブロックを持つ

### 防御系

#### 盾カード

- 配置はOカードと同じ
- コスト1, 攻撃力0
- 全マスに盾ブロックを持つ

### 補助系

#### マナカード

- 配置はTカードと同じ
- コスト2, 攻撃力0
- 中心1マスにマナブロックを持つ

#### 剣カード

- 配置はIカードと同じ
- コスト1, 攻撃力2
- 全マスに剣ブロックを持つ

#### ドローカード

- 配置は縦3マス
- コスト2, 攻撃力0
- 中心1マスにドローブロックを持つ

#### 手痛い増資

- 配置は縦3マス
- コスト1, 攻撃力1
- トゲ、ドロー、トゲを持つ

## 特殊ブロック

| 種別           | 効果                                                                |
| -------------- | ------------------------------------------------------------------- |
| 爆弾ブロック   | 消滅時に周囲8マスを消しつつ敵全体に10追加ダメージを与える           |
| 剣ブロック     | 配置している間中、全てのミノカードの配置時の基礎ダメージが1上昇する |
| 盾ブロック     | 配置時に次の敵ターンに受けるダメージを1軽減する                     |
| マナブロック   | 消滅時にMPが1回復する                                               |
| ドローブロック | 配置時に手札を1枚引く                                               |
| トゲブロック   | ターン終了時に盤面に存在する場合、自分に1ダメージ与える             |
